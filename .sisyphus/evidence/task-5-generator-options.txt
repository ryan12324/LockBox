IMPLEMENTATION NOTES
====================

1. Random Number Generation:
   - Uses crypto.getRandomValues() for all randomness
   - Helper function getRandomIndex(max) ensures proper modulo distribution
   - No reliance on Math.random() anywhere in source

2. Password Generation:
   - Builds character pool from enabled sets
   - Uses rejection sampling approach to ensure at least one char from each set
   - Supports ambiguous character exclusion (0O1lI|)

3. Passphrase Generation:
   - Selects random words from WORDLIST using crypto.getRandomValues()
   - Supports optional number inclusion
   - Flexible separator and capitalization options

4. Strength Evaluation:
   - Leverages @zxcvbn-ts/core for realistic scoring
   - Provides entropy in bits (converted from log10 scale)
   - Includes actionable feedback for users

5. Word List:
   - ~1000 common English words
   - Suitable for memorable passphrases
   - Embedded inline to avoid external dependencies
