🛡️ Security Features

2FA detection badge, phishing warnings, and WebAuthn interceptor.

2FA Detection badge

After autofilling a login for a site that supports 2FA (e.g. github.com), a 2FA recommendation badge should appear in the bottom-right corner.

  • Autofill a login item for a 2FA-supported site
  • Badge appears in bottom-right with indigo gradient
  • Shows site name, "Enable 2FA" message
  • Link to 2FA setup guide (if available)
  • Auto-dismisses after 15 seconds
  • Dismiss button works

WebAuthn / Passkey interceptor

The extension intercepts navigator.credentials.create/get. Click to trigger a passkey registration attempt.

📋 Manual Test Checklist

Things to verify manually across test scenarios.

Lock Icon Overlay

  • Lock icon visible on all password fields
  • Lock icon positioned correctly (not clipped)
  • Lock icon click opens autofill/suggestion dropdown
  • Icon doesn't break field layout or overlap text
  • Icon appears on dynamically injected forms
  • Identity forms get lock icon on first field

Autofill Dropdown

  • Single match → auto-fills immediately
  • Multiple matches → shows dropdown with account list
  • Dropdown shows name + username for each item
  • Clicking item fills both username and password
  • Dropdown dismisses on outside click
  • Identity dropdown shows identity names

Save-on-Submit Banner

  • Banner appears at page top after form submit
  • Shows "Save" and "Dismiss" buttons
  • "New credential" vs "Update" distinction
  • Banner uses Shadow DOM (no CSS conflicts)
  • Auto-dismisses after 30 seconds
  • Works with AJAX/fetch submissions

Popup Features

  • Current Site tab matches this page
  • Vault tab shows all items
  • Generator tab produces passwords
  • Smart generation detects field rules
  • TOTP codes display and auto-copy
  • Shared tab shows team items
  • Health tab shows vault analysis
  • Trash view lists deleted items
  • Settings view has 2FA / travel mode
  • Emergency access view works
  • Version history shows in item detail
  • HW keys view lists / registers keys
  • QR sync generates / scans codes
